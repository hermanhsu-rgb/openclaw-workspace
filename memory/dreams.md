## 📖 关于梦境记忆

梦境记忆是 OpenClaw 2026.4.5 引入的实验性功能，通过以下三个阶段在后台运行：

### 三阶段梦境流程

1. **Light（浅层）** - 快速扫描近期记忆，识别关键模式
2. **Deep（深层）** - 分析重要对话，提取持久性洞察
3. **REM（快速眼动）** - 生成概念性关联，创建长期记忆

### 配置参数

| 参数 | 值 | 说明 |
|------|-----|------|
| `enabled` | true | 已启用 |
| `frequency` | daily | 每日运行 |
| `recencyHalfLifeDays` | 7 | 近期记忆半衰期 |
| `maxAgeDays` | 30 | 最大记忆保留天数 |
